# Brand Management System

## Quick Start

1. **Create Database:**
   ```sql
   CREATE DATABASE brand_management_db;
   ```

2. **Open in IntelliJ:** File → Open → `C:\Users\ADMIN\Downloads\BrandManagementSystem`

3. **Run:** `BrandManagementApplication.java`

4. **Open Browser:** `http://localhost:8080`

## Key Fix

Table `groups` renamed to `customer_groups` (MySQL reserved keyword fix).

## Features
- Dashboard with stats
- Add/Edit/Delete brands
- Filter by Group or Company
- Soft delete (activate/deactivate)
